# Данные
Файлы кейса лежат в репозитории (готовы после clone):
- `codebase/` - репозиторий `gymhero` (распакованный `codebase_python.zip`)
- `eval_questions.json` - вопросы для оценки
- `sample_queries.txt` - примеры запросов
- `score.py` - официальный скорер (Precision@5/Hit@5)

Как пересобрать из исходного архива - в [корневом README](../README.md).
